說明文件
data-verify="name" 驗証姓名
data-verify="mail" 驗証mail
data-verify="mobile" 驗証手機
data-verify="password" 驗証密碼
data-verify="same" 驗証相同欄位 (將需要被一致的欄位 新增一個與data-same裡的值 一樣的classname)
例：data-same="same" 需要被相同的欄位上則是加上 class="same"
data-msg 錯誤訊息文字
填在子層 訊息會顯示在 所填的tag裡
填在父層(與驗証data-name)同一層裡 則會在子層建立一個p 將錯誤訊息顯示在p 裡
data-submit 送出 
      $('[data-form]').validate({
        success: function(){
          // 送出成功後 所要執行的事件
        },
        error: function(){
          // 送出失敗後 所要執行的事件
        }
      });
